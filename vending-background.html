How to Use This Background

This page displays the vending machine background in a responsive viewer. You can:

  • View the design - See how the background will look on your vending machine screen
  • Save as Image - Click the button above to create a downloadable image
  • Upload to Vending Machine - After downloading, transfer to USB drive for upload

Uploading to Your KoolMore Vending Machine

  1. Create a folder named KMFolder on your USB drive
  2. Inside that folder, create another folder named ImageGoods
  3. Copy the downloaded image into the ImageGoods folder
  4. Insert the USB drive into the vending machine's USB port
  5. Access the Android system menu and use the "Backup ads from flash driver" option
  6. Restart the machine to apply the changes
